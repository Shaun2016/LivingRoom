## 模仿B站样式的直播间

*使用 Bootstrap 前端框架*

### 功能

* 可以发弹幕，同时显示在弹幕列表中
* 弹幕列表可以打开关闭

### 问题

* 弹幕列表列滑动时无法冻结表头
* 关闭弹幕对新发送的弹幕无法隐藏
* 弹幕列表展示时由于修改高度产生闪动

### 访问

> **打开 room.html**

### 文件结构

* css
  * bootstrap.min.css
  * my.css    自定义的样式文件
* js
  * bootstrap.min.js
  * jquery-3.3.1.js
  * live.js    弹幕发送事件等
